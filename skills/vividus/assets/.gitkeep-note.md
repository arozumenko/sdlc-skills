# assets/

Copy-able templates the skill uses to scaffold a Vividus project quickly.

| File | Destination in target project |
|---|---|
| `build.gradle.template` | `build.gradle` (project root) |
| `gradle.properties.template` | `gradle.properties` (project root) |
| `configuration.properties.template` | `src/main/resources/properties/configuration.properties` |
| `suite.properties.template` | `src/main/resources/properties/suite/<name>/suite.properties` |
| `profile.web-chrome.properties.template` | `src/main/resources/properties/profile/web/desktop/chrome/profile.properties` |
| `environment.properties.template` | `src/main/resources/properties/environment/<env>/environment.properties` |
| `example-web.story` | `src/main/resources/story/web_app/Login.story` (or any name) |
| `example-rest.story` | `src/main/resources/story/rest_api/Star Wars API.story` |
| `example.steps` | `src/main/resources/steps/<area>/<name>.steps` |

These are starting points — bump the BOM version, fill in real URLs/credentials, and prune what you don't need.
